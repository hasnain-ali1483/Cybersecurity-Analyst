echo : Output any text that we provide
Example: echo "Hello World"

whoami: Output the username of the current user
Example: whoami

clear: Clear the screen
Example: clear

sleep: Pause the terminal for a specified amount of time
Example: sleep 5
         sleep 10m
         sleep 1h

wc : Word count
Example: wc file1.txt
         wc -l file1.txt (Gives the number of lines in the file)
         wc -c file1.txt (Gives the number of byes in the file)
         wc -w file1.txt (Gives the number of words in the file)

Interacting With the Filesystem:

 ls : List the contents of the current directory
 Example: ls

 cd : Change directory
 Example: cd folder4

 cat : Display the contents of a file
 Example: cat file1.txt

 pwd : Print the current working directory
 Example: pwd

 touch : Create a new file
 Example: touch file1.txt

 mkdir: Create a new directory
 Example: mkdir folder1

 rm : Remove a file or directory
 Example: rm file1.txt
          rm -r folder1 (for removing a directory)

 cp : Copy a file or directory
 Example: cp file1.txt file2.txt (Copying from file1.txt to file2.txt, if file2.txt does not exist, it will be created)

 mv : Move a file or directory
 Example: mv file1.txt file2.txt (Moving from file1.txt to file2.txt, if file2.txt does not exist, it will be created)
          mv file1.txt folder1

 file : Display information about a file
 Example: file file1.txt

Searching for Files:

 find : Search for files
 Example: find -name "*.txt" (Searches for all files with the .txt extension)
          find -name "file1.txt"

 grep : Search for text in files
 Example: grep "Hello" file1.txt

Shell Operators:

 & : Run a command in the background (job ID and process ID (PID) gets printed too).
 Example: ls &
          sleep 5 & echo "Hello World"

 && : This allows you to combine multiple commands together in one line of your terminal.
 Example: echo "Hello World" && echo "Goodbye World"

 > : Redirect the output of a command to a file
 Example: echo "Hello World" > file1.txt

 >> : This does the same function of the > operator but appends the output rather than replacing (meaning nothing is overwritten).
 Example: echo "Hello World" >> file1.txt


Flags and Switches:
A majority of commands allow for arguments to be provided. These arguments are identified by a hyphen and a certain keyword known 
as flags or switches.

 -a : This argument (short for --all) will display all files(including hidden files) in the current directory.
 Example: ls -a

 --help : This option will list the possible options that the command accepts, provide a brief description and example of how to use it.
 Example: ls --help

 man : Display the manual page for a command.
 Example: man ls
